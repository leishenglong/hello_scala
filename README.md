# hello_scala

[![Build Status](https://travis-ci.org/PhonyLou/hello_scala.svg?branch=master)](https://travis-ci.org/PhonyLou/hello_scala)

A project to show demo code from learning <I>Programming in Scala</I>

## Prerequisites

- OpenJDK8
- Scala 2.12+
- SBT 1.3+

## How to run?

```shell
$ sbt clean test
```
